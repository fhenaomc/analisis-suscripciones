\# Análisis de suscripciones y abandono
Proyecto de análisis de datos enfocado en suscripciones, churn y manejo de datos incompletos.

Sobre la empresa: Comercializa suscripciones para servicio de streaming 

Problemática: Analizar información incompleta y no relacionada directamente con el churn. En general y  en la mayoría de los casos el costo de un cliente nuevo es superior al de mantener un cliente activo, así que ignorar esta problemática supone un riesgo potencialmente alto para el negocio.

Retos del caso: Se tienen datos básicos de los usuarios (ID de usuario, edad, ubicación, forma de pago, tipo de suscripción...) y datos de uso (tiempo de uso, registros de conexión, gustos dentro de la plataforma, elementos guardados en favoritos...) lo cual ayuda a construir los diferentes perfiles de los usuarios, pero no se tienen encuestas de satisfacción o de percepción de valor. El reto se fundamenta en explorar las posibles señales asociadas al churn y evaluar qué variables parecen influir de manera más directa en la desuscripción de la plataforma. 

Alcance: 
\- No se pretende construir un perfil unificado de usuario con alto riesgo de abando de plataforma ni un modelo predictivo para este mismo fin. 
\- Debido al tipo de datos base, no se concluirá con base en la percepción de los usuarios respecto a la plataforma ni nada que no se tenga de manera explícita en los datos o que estén directamente relacionados a estos (a menos que se puedan hacer inferencias soportadas por otros estudios).
\- Para un análisis más completo de churn sería necesario realizar encuestas directamente dirigidas a los usuarios.
